# 🚀 InternHub

InternHub is a modern internship portal designed to connect students with internship opportunities across various domains. The platform provides an intuitive interface for browsing internships, viewing details, posting opportunities, and managing user profiles.

## 🌐 Live Demo

**Website:** https://internhub-wine.vercel.app/

## 📂 GitHub Repository

https://github.com/Mahek-rgb/internhub

---

## ✨ Features

- 🏠 Modern and responsive homepage
- 🔍 Search interface for internships
- 💼 Internship listings page
- 📝 Internship application page
- 👤 User profile page
- 🔐 Admin/Login page
- 📢 Post Internship page
- 📱 Responsive design for desktop and mobile
- 🎨 Clean and user-friendly interface

1. Clone the repository

```bash

```

2. Open the project folder

3. Open `index.html` in your browser

No additional installation is required.
